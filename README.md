# Asper-task-4--choice-based-storyGame-
This is a mini Javascript project.
Choice-Based Mystery Game

A simple interactive story game where every decision changes the path of the story.

Each scene gives you two choices, and every choice leads to a different outcome. Explore different paths, uncover clues, and reach multiple endings based on your decisions.

🔍 Theme

Dark, psychological, murder-mystery stories where every character can be suspicious and every choice matters.

🎮 How to Play
Select a story
Click Start
Choose between the options
See where your decisions lead

Restart and try different paths to discover all possible endings.

Every decision has a price.
